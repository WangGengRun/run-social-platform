# 知识库：平台使用指南与账号管理

## 1. 平台简介
校友社交平台是一个专为本校校友设计的私密、安全的在线社区。我们的目标是帮助校友重建联系、共享资源、促进职业发展，并维系与母校的情感纽带。平台提供校友档案、动态分享、实时聊天、活动组织和AI智能助手等核心功能。

## 2. 账号注册与登录
-   **注册方式**：新用户需要使用常用手机号或邮箱进行注册。
-   **身份验证**：为确保社区的真实性与安全性，注册后需进行校友身份验证。请在“个人中心”->“校友认证”页面，上传您的学生卡、毕业证或学位证的照片。管理员将在1-3个工作日内完成审核。
-   **登录**：注册成功后，可使用手机号/邮箱和密码进行登录。平台支持“记住密码”功能，方便下次快速登录。
-   **忘记密码**：在登录页面点击“忘记密码”，通过注册时绑定的手机号或邮箱接收验证码，即可重置密码。

## 3. 账户安全与设置
-   **修改密码**：登录后，在“个人中心”->“账户安全”中，输入旧密码和新密码即可完成修改。建议定期更换密码，并避免使用过于简单的组合。
-   **绑定信息**：您可以在账户设置中更新绑定的手机号和邮箱地址，以确保能正常接收系统通知和活动提醒。
-   **隐私设置**：您可以控制个人档案信息的可见范围，例如，设置手机号、邮箱仅对自己可见，或仅对好友/同城校友可见。前往“个人中心”->“隐私设置”进行调整。

## 4. 常见问题 (FAQ)
-   **问：为什么我无法发布动态或报名活动？**
    -   **答：** 大部分社交和活动功能需要您先通过“校友身份验证”。请检查您的认证状态是否为“已通过”。
-   **问：我上传了认证照片，但很久没有审核结果？**
    -   **答：** 管理员通常会在1-3个工作日内完成审核。如果超过3天仍未处理，请联系平台客服邮箱或通过“意见反馈”功能咨询。
-   **问：我收不到短信验证码怎么办？**
    -   **答：** 请先确认您注册的手机号是否正确，并检查手机是否开启了短信拦截功能。若无问题，可尝试1分钟后再次获取，或更换为邮箱验证方式。
-   **问：如何注销我的账号？**
    -   **答：** 出于数据安全考虑，目前暂不支持在线注销。如需注销账号，请联系平台管理员，并提供必要的身份证明进行人工处理。
