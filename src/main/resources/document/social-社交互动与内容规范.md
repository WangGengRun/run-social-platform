# 知识库：社交互动与内容规范

## 1. 核心社交功能
-   **发布动态**：在“动态”页面，您可以发布图文动态，分享近况、职业成就、行业见解或校园回忆。支持上传多张图片。发布时，可以选择动态的可见范围，如“所有人可见”或“仅好友可见”。
-   **互动交流**：您可以对他人的动态进行“点赞”和“评论”。所有评论都是公开的，请注意文明用语。您也可以回复他人的评论，形成楼中楼对话。
-   **关注机制**：关注您感兴趣的校友后，他们的新动态会优先出现在您的“关注动态”流中。这能帮您过滤信息，高效关注重点人脉。
-   **实时私信**：点击他人个人主页上的“私信”按钮，或进入“消息”中心，可以开始一对一的实时聊天。私信功能基于WebSocket技术，能保证消息的即时送达。

## 2. 平台内容规范
为了维护一个健康、积极、文明的社区环境，请遵守以下规范：
-   **禁止内容**：严禁发布任何形式的广告、谣言、人身攻击、歧视言论、政治敏感内容、色情暴力信息或侵犯他人隐私的内容。
-   **友善互动**：鼓励理性、客观、友善的讨论。反对网络暴力、引战和恶意刷屏。
-   **版权尊重**：转载他人文章或图片时，请注明出处或确保已获得授权。
-   **违规处理**：对于违反规范的内容和用户，管理员有权进行警告、内容删除、限制功能直至封禁账号等处罚。

## 3. 举报与反馈
如果您在平台上发现了违规内容或骚扰行为，请及时使用“举报”功能。平台管理员会尽快核实并处理。您也可以通过“意见反馈”通道，对平台功能提出改进建议。

## 4. 常见问题 (FAQ)
-   **问：我发布的动态可以删除吗？**
    -   **答：** 可以。在您自己发布的动态下方，有一个“删除”按钮，点击即可删除该条动态及所有点赞和评论。
-   **问：如何屏蔽某个用户的骚扰？**
    -   **答：** 在对方的个人主页上，点击“更多”菜单，选择“加入黑名单”。此后，您将不再收到对方的消息和动态。
-   **问：我收到了一条违规私信，该怎么办？**
    -   **答：** 请立即截屏保留证据，并在私信界面选择“举报”。管理员会介入调查。
