# 知识库：活动中心指南

## 1. 浏览与报名活动
-   **活动广场**：在“活动中心”页面，您可以查看所有正在招募、即将开始和已结束的活动。活动类型包括线上分享会、线下同城聚会、返校日活动、企业参访等。
-   **活动详情**：点击活动卡片，进入详情页，可以查看活动时间、地点、人数限制、活动流程和注意事项等完整信息。
-   **在线报名**：如果您对某个活动感兴趣，点击“立即报名”按钮即可。系统会实时校验名额，成功后您将收到站内信或邮件通知。
-   **取消报名**：如果计划有变，请在报名截止时间前进入“我的活动”页面，找到相应活动并点击“取消报名”，以便将名额留给其他校友。

## 2. 活动签到
对于需要线下签到的大型活动，组织者可能会提供二维码签到。您可以在“我的活动” -> “待参与”中找到该活动，点击“出示签到码”，由现场工作人员扫描完成签到。签到记录有助于组织者统计人数和提升后续活动体验。

## 3. 创建与发布活动
-   **权限**：目前，活动的创建权限主要开放给校友会官方、院系管理员或经过认证的班级联络人。普通校友如需发起活动，可以联系管理员申请权限。
-   **创建流程**：拥有权限的用户，在“活动中心”页面点击“创建活动”，填写活动标题、类型、地点、起止时间、报名截止时间、人数限制和详细描述后提交。管理员审核通过后，活动即会公开上线。

## 4. 常见问题 (FAQ)
-   **问：报名后如何获取活动的最新通知？**
    -   **答：** 活动开始前，系统会通过站内信和邮件向所有报名成功的校友发送提醒通知。请确保您的联系方式是最新的。
-   **问：活动名额已满，我还能报名吗？**
    -   **答：** 如果报名人数已达到上限，报名按钮会变为“名额已满”。您可以关注该活动，如果有人取消报名，系统可能会释放名额。部分活动会开放“候补名单”，您可以加入候补，有机会递补。
-   **问：我错过了活动的签到，会有什么影响吗？**
    -   **答：** 偶尔错过签到影响不大。但如果多次报名活动后无故缺席（No-Show），可能会影响您后续报名其他热门活动的优先级。请珍惜每一次参与机会。
-   **问：我想组织一个本年级的聚会，如何申请成为活动发起人？**
    -   **答：** 请通过“联系客服”功能，发送您的校友身份信息和活动初步计划，平台管理员会与您进一步沟通。
