# 知识库：职业发展与校友资源

## 1. 寻找与连接校友
-   **精准搜索**：在“校友档案”页面，您可以根据“入学年份”、“学院”、“专业”、“所在城市”、“所属行业”和“公司名称”等多个维度进行组合筛选，快速定位您想找的校友。
-   **查看档案**：点击搜索结果中的校友姓名，可以访问其个人主页，查看其公开的教育背景、工作履历和自我介绍。
-   **建立联系**：在对方个人主页上，您可以点击“关注”成为其粉丝，后续能在“我的关注”动态流中看到对方的公开动态。如果需要深入交流，可以使用“私信”功能直接沟通。

## 2. 资源与机会
-   **招聘与求职**：平台鼓励校友发布内推和招聘信息。您可以在“校友动态”或专门的“招聘求职”版块（若有）查看相关岗位。发布招聘信息时，请详细描述岗位职责、要求和联系方式。
-   **寻求建议**：AI助手可以为您提供职业发展方面的建议。例如，您可以提问：“我是计算机专业的大三学生，未来想做人工智能，应该学习哪些核心课程？”或“如何准备产品经理的校招面试？”。AI会基于知识库提供通用性指导。
-   **导师计划**：关注平台公告，学校可能会发起“校友导师计划”，邀请资深校友为在校生和年轻校友提供职业咨询。届时可通过活动中心报名。

## 3. 个性化推荐
AI助手会根据您的个人档案（如专业、毕业年份、行业）和您在平台上的行为，主动为您推荐可能感兴趣的校友、动态或活动。例如：
-   “推荐关注：有3位同是2008级计算机学院的校友最近加入了平台。”
-   “猜你喜欢：来自您所在城市的校友张华发布了一则关于人工智能行业分享的动态。”

## 4. 常见问题 (FAQ)
-   **问：我搜索一个校友，但没有找到他/她，怎么办？**
    -   **答：** 可能该校友尚未注册平台，或者其隐私设置导致您无法搜到。您可以尝试通过其他已知的校友帮忙转告，或在“找校友”板块发布寻人帖（需注意保护个人隐私）。
-   **问：AI助手能直接帮我联系到某位校友吗？**
    -   **答：** 不能。AI助手无法执行任何侵犯用户隐私的操作，如获取未公开的联系方式或代发私信。它只能基于公开信息和知识库为您提供寻找或接触校友的建议。
